# Welcome to SnippetX

- This is simple and minimal extension alternative to ES7+ React/Redux/React-Native snippets
- No unnecessary snippets.
- Only useful snippets.

## Features

0. ### JavaScript Snippets

   - clg
   - clo
   - cls
   - jmap
   - jfilter
   - jsfy
   - jfind
   - jparse

1. ### React Snippets

   - rfc
   - rfcpe
   - us
   - ue
   - od
   - rmap
   - rfind
   - rfilter

2. ### Redux-Toolkit Snippets

   - slice
   - extraReducers
   - addMatcher
   - thunk
